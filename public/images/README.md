# Carpeta de Imágenes

Coloca aquí las siguientes imágenes para que se muestren correctamente en la web:

## Imágenes Requeridas

1. **logo.png** (o logo.svg)
   - Logo del estudio notarial
   - Recomendado: formato PNG con fondo transparente
   - Tamaño sugerido: mínimo 400x400px para alta calidad

2. **hero-background.jpg**
   - Imagen de fondo para la sección Hero
   - Recomendado: formato JPG optimizado
   - Tamaño sugerido: mínimo 1920x1080px
   - Debe ser una imagen profesional (escritorio, documentos, ambiente notarial)

## Notas

- Todas las imágenes deben estar optimizadas para web
- Usa formatos modernos (WebP) cuando sea posible para mejor rendimiento
- Las imágenes se cargan automáticamente desde esta carpeta usando Next.js Image component
- Si alguna imagen no está disponible, se mostrará un placeholder

## Estructura de Archivos

```
public/
  images/
    logo.png          (o logo.svg)
    hero-background.jpg
```
